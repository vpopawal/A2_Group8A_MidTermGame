## Description

Brief overview of your game’s concept, mechanics, and player experience

Pixelated Thoughts is narrative-driven interactive game designed to rasie awareness about Schizophrenia and helps players beter understand the challenges individuals with the condition may experince.The game simulates disorganized thinking and intrusive thoughts gameplay mechanices that intentionally disrupt the players focus.

Players navigate a series of social interactions where they must assemble sentences from scrambled word fragments to communicate with others. However, random thought interruptions appear on the screen, forcing the player to clear them before continuing. Some word fragments disappear quickly, creating time pressure and simulating the diffculty of organizing thoughts in conversation.

Through these mechanics the game aims to create empathy and awareness by giving players a small glimpse into how overwhleming and confusing everyday communication can feel for someone experienceing symptoms of Schizophrenia.

## Interaction Instructions

How to Run the Game:

1. Download or clone the project folder.
2. Open the project in VS Code or any code editor.
3. Run the project using Live Server or open the index.html file in a browser.
4. The game will load in the browser window.
5. The game uses the p5.js library for rendering graphics and handling interaction.

How to play the game:

1. Start Screen: Press SPACE to start the game
2. Character selection: Press 1, Press 2, Press 3 to choose a character
3. Exploring the world: move around with Arrow keys to walk around the building
4. When the enter prompt appear as the character is near door entrance. Press “ENTER” to go inside the store

Level Selection:

1. Each building has 3 levels.
2. Levels unlock progressively as you complete them.

Inside the Store:

- Read the question in the speech bubble.
- Click the correct words in order to build the sentence.
- Words move around the screen.
- Some words are distractors

Distractions

- Intrusive thoughts appear randomly on screen and can be clicked away.
- A timer counts down during the level.

Failure Conditions

- Clicking the wrong word
- Timer running out
- If you fail, you can retry or return to the world map.

## Interation Notes

Post-Playtest :
Three changes made based on playtesting

Added Level Lock Feature: Players were jumping ahead to harder sub-levels before completing the earlier ones, which disrupted the intended difficulty curve and narrative flow. A lock system was implemented so that each sub-level within a store is locked until the previous one is completed. Locked levels are visually indicated with a padlock icon on the level select screen.

Fixed Sentences and Game Mechanics: During playtesting, several sentence targets were found to be unclear or had issues with word matching logic (e.g., capitalization inconsistencies and incorrect win condition checks). The sentences across all four stores were reviewed and corrected based on matching each store's specific items and edited such that in moves hard every level.

Added Return Home Button and X Exit Button: Playtesters told us about feeling "stuck" on the fail screen with no intuitive way to navigate back. A "Return to World" button was added to the fail screen.

A red X exit button was added on the right top of the in-store, allowing players to exit a level and return to the level select screen at any time without having to fail or complete the level.

Post-Showcase: Planned improvements

- A visual indication that you win a level.
- Use wasd key instead of the arrow keys.
- Do more research on schizophrenia on how exactly we can improve in our game make it more obvious.
- Improve the cognitive perspective like adding constraints on the houses or having a clear path the player can use to walk on.
- Make instructions more clear when playing the game and have instructions tab open when needed.
